# Fruit Quality Detection System using Convolutional Neural Networks (CNN)

## Overview

The Fruit Quality Detection System is a machine learning-based project that uses Convolutional Neural Networks (CNN) to identify and classify the quality of fruits. The system helps in detecting whether a fruit is fresh or rotten by analyzing fruit images.

This project demonstrates the application of Deep Learning and Computer Vision in agriculture and food quality inspection.

---

## Features

* Fruit image classification using CNN
* Detects fruit quality (Fresh / Rotten)
* User-friendly interface
* Deep learning model training and prediction
* Image preprocessing and feature extraction
* Helps reduce manual inspection effort

## Installation

### Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/Fruit-Quality-Detection-System.git
```

### Step 2: Navigate to Project Folder

```bash
cd Fruit-Quality-Detection-System
```

### Step 3: Install Dependencies

```bash
pip install -r requirements.txt
```

## Run the Project

